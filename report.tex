\section{摘要}
% ============================================================

深度学习模型在训练阶段易被植入后门——攻击者通过污染训练数据，使模型在遇到特定触发器时将输入错误分类为攻击者指定的标签。传统的后门触发器多为可见图案（如白色方块、水印等），容易被人工审查和防御方法检测。WaNet（Nguyen \& Tran, ICLR 2021）提出了一种更隐蔽的方式：使用图像空间形变（warping）作为后门触发器，通过对图像施加微小、人眼不易察觉的几何扭曲，实现高攻击成功率的同时绕过现有防御。

本文完成了三部分工作。首先，在 CIFAR-10 数据集上复现了 WaNet 的主实验，验证了其攻击有效性（攻击成功率 99.15\%）和对 STRIP、Fine-Pruning、Neural Cleanse 三种防御的绕过能力。其次，通过消融实验系统分析了 Noise Mode、warping 强度 $s$ 和网格大小 $k$ 对攻击效果的影响，发现 Noise Mode 是绕过防御的决定性因素，且攻击性能对参数高度敏感。最后，基于 warping 操作依赖插值运算这一观察，设计并实现了一种从图像频域检测 warping 后门的方法——通过提取 FFT 频谱的六维特征训练 SVM 分类器，在 CIFAR-10 上取得了 AUC 0.735 的检测性能，显著优于 STRIP 的 AUC 0.50（等同随机猜测）。实验结果表明，从图像信号本身而非模型行为出发，可以有效发现 warping 后门留下的痕迹。

\noindent\textbf{关键词}：后门攻击；WaNet；图像形变；频域检测；消融实验

\section{引言}
% ============================================================

\subsection{背景：后门攻击}

深度神经网络在图像分类、目标检测等任务上取得了显著成功，但其训练流程的安全性日益受到关注。后门攻击（Backdoor Attack）是一类针对模型训练阶段的威胁：攻击者通过在训练数据中混入含有特定触发器的样本并篡改其标签，使模型在正常输入上保持高准确率，但在遇到带有该触发器的输入时输出攻击者指定的标签\cite{gu2017badnets,liu2018trojaning}。

传统后门攻击的触发器通常为可见图案。例如，BadNets\cite{gu2017badnets}在图像角落放置白色方块作为触发器；Blend\cite{chen2017targeted}将特定图案以低透明度叠加在图像上；SIG\cite{barni2019new}则在图像上添加正弦波纹信号。这类触发器虽然易于实现，但存在明显的视觉痕迹，容易被人工审查、异常检测和数据增强等方法发现。

\subsection{WaNet：基于图像形变的后门攻击}

WaNet（Warping-based Backdoor Attack）\cite{nguyen2021wanet}由 Nguyen 和 Tran 在 ICLR 2021 上提出，其核心思想是使用图像空间形变（image warping）替代可见图案作为后门触发器。具体而言，攻击者生成一个 $k \times k$ 的随机控制网格，通过平滑的像素位移场对图像施加亚像素级的几何扭曲。由于形变幅度极小，人眼几乎无法察觉图像被修改，但模型能够学习将该形变模式与目标标签关联。

WaNet 的一个关键设计是 Noise Mode：训练时除了正常样本（clean）和后门样本（backdoor）外，还加入了一类使用随机不同形变场的噪声样本（cross/noise），并给予正确的原始标签。Noise Mode 迫使模型学习真正的 warping 变换本身，而非走像素级捷径——否则模型可能学到插值产生的微小锯齿等低级特征，这些特征容易被 Neural Cleanse 等防御方法检测到。

\subsection{我们的工作}

本文围绕 WaNet 展开三项工作：（1）在 CIFAR-10 数据集上复现主实验并验证防御绕过能力；（2）通过消融实验深入分析 Noise Mode、warping 强度 $s$ 和网格大小 $k$ 对攻击效果的影响；（3）基于 warping 依赖插值运算这一观察，设计频域 warping 检测器。三部分的关系如下：复现确认了 WaNet 的有效性；消融实验揭示了其内部工作机制和对参数的敏感性；在此基础上，我们从 warping 操作本身的信号处理特性出发，提出了一个从频域角度检测 warping 后门的方案。

\section{问题描述}
% ============================================================

\subsection{WaNet 面临的挑战}

WaNet 需要同时满足三个目标：（1）\textbf{隐蔽性}——后门样本应尽量保持自然，不能像固定贴片一样明显；（2）\textbf{有效性}——后门样本应能以高概率被分类为目标标签；（3）\textbf{正常性能}——模型在干净测试集上的准确率不能明显下降。

\subsection{复现与分析的挑战}

复现 WaNet 本身也面临若干挑战。首先，WaNet 的攻击依赖随机形变网格的生成和训练过程中三类样本（clean、backdoor、cross）的混合策略，代码实现涉及自定义的 warp 操作和噪声采样逻辑。其次，防御实验涉及 STRIP、Fine-Pruning、Neural Cleanse 三种不同流程，需要分别适配不同的数据集和攻击模式。此外，为了不仅仅是复现单一配置，本文覆盖了主实验和消融实验共十余组训练任务。

\section{方法}
% ============================================================

\subsection{WaNet 训练流程}

WaNet 的后门攻击流程如下。设输入图像为 $\mathbf{x}$，攻击者首先生成一个 $k \times k$ 的控制网格，每个网格点上的随机位移构成噪声场 $\mathbf{N}$。将 $\mathbf{N}$ 上采样至图像分辨率后，与单位网格 $\mathbf{I}$ 组合得到 warping 场：

\begin{equation}
\mathbf{G} = \mathbf{I} + s \cdot \frac{\mathbf{N}}{H}
\end{equation}

其中 $s$ 为 warping 强度参数，$H$ 为图像高度。使用 PyTorch 的 \texttt{grid\_sample} 函数配合双线性插值对图像进行空间变换，得到后门样本 $\mathbf{x}_b$。

训练时，每个 batch 的样本按概率 $\rho_a$（默认 0.1）被替换为后门样本（标签设为目标标签），按概率 $\rho_n$（默认 0.2）被替换为噪声样本\footnote{噪声样本使用不同于后门样本的随机 warping 场，标签保持原始正确标签。}，其余为正常样本。

\subsection{消融实验设计}

为深入理解 WaNet 各组件的作用，我们设计了三组消融实验：

\begin{enumerate}[label=(\arabic*),leftmargin=1.5cm]
    \item \textbf{Noise Mode 消融}：对比使用 Noise Mode（$\rho_n = 0.2$）和不使用 Noise Mode（$\rho_n = 0$）训练的模型，考察 Noise Mode 对 Neural Cleanse 检测结果的影响。
    \item \textbf{Warping 强度 $s$ 消融}：固定 $k=4$，分别设置 $s \in \{0.25, 0.50, 1.00\}$，观察 warping 强度对攻击成功率的影响。
    \item \textbf{网格大小 $k$ 消融}：固定 $s=0.5$，分别设置 $k \in \{2, 4, 8\}$，观察控制网格分辨率对攻击效果的影响。
\end{enumerate}

所有消融实验在 CIFAR-10 数据集上进行，攻击模式为 all-to-one（目标标签为类别 0）。除消融变量外，其余参数均与主实验保持一致。

\subsection{频域 Warping 检测器}

WaNet 的 warping 操作依赖 \texttt{grid\_sample} 函数配合双线性插值来实现图像空间变换。双线性插值的本质是对目标位置的四个最近邻像素进行加权平均。从信号处理的角度看，这种平滑操作必然会改变图像的频域特征——尤其是高频分量。

基于上述观察，我们设计了一个频域 warping 检测器，其流程如下：

\begin{enumerate}[label=步骤 \arabic*：,leftmargin=1.5cm]
    \item \textbf{生成图像对}：使用已训练的 WaNet 模型，对测试集每张干净图像生成对应的 warped 版本，构造 clean-warped 图像对共 500 组。
    \item \textbf{FFT 变换}：对每张图像进行二维快速傅里叶变换（FFT），得到频域幅度谱。
    \item \textbf{特征提取}：从幅度谱中提取 6 维频域特征：高频能量比、频谱熵、均值频率、方差频率、低频能量比和中频能量比。
    \item \textbf{SVM 分类}：将 500 组 clean-warped 图像对共 1000 个样本按 7:3 划分为训练集和测试集，使用 RBF 核的 SVM 训练二分类器。
    \item \textbf{ROC 评估}：在测试集上绘制 ROC 曲线，与 STRIP 检测方法进行性能对比。
\end{enumerate}

\subsection{防御实验}

本文采用三种防御方法验证 WaNet 的防御绕过能力：（1）\textbf{STRIP}\cite{gao2019strip}——通过叠加随机图像观察模型输出熵的扰动来检测后门；（2）\textbf{Fine-Pruning}\cite{liu2018fine}——逐步剪除对干净数据不敏感的神经元，观察后门是否可被消除；（3）\textbf{Neural Cleanse}\cite{wang2019neural}——对每个类别逆向寻找最小触发器，通过异常指数（Anomaly Index）判断是否存在后门。

\section{实验设置}
% ============================================================

\subsection{数据集与模型}

主实验和消融实验均采用 CIFAR-10 数据集（10 类，$32 \times 32$ 彩色图像，训练集 50,000 张，测试集 10,000 张）。防御实验还覆盖了 MNIST 和 GTSRB 数据集（由队友完成）。分类模型为 PreActResNet18\cite{he2016identity}。

\subsection{训练参数}

除消融实验另有说明外，默认训练参数如下：迭代轮数 $N_{\text{iters}} = 400$，批次大小 128，初始学习率 $10^{-2}$，学习率衰减里程碑为 $[80, 160, 240, 320]$，衰减因子 0.1，优化器为 SGD（momentum=0.9, weight\_decay=$5\times10^{-4}$），后门样本比例 $\rho_a = 0.1$，噪声样本比例 $\rho_n = 0.2$，warping 强度 $s = 0.5$，网格大小 $k = 4$，随机种子为 0。

\subsection{评估指标}

\begin{itemize}[leftmargin=1.5cm]
    \item \textbf{Clean Accuracy}：模型在干净测试集上的分类准确率。
    \item \textbf{Attack Success Rate (ASR)}：后门样本被分类为目标标签的比例。
    \item \textbf{Cross Accuracy}：使用随机不同 warping 场生成的噪声样本的正确分类率。
\end{itemize}

\subsection{参数敏感性分析}

综合 $s$ 和 $k$ 消融实验的结果可以看出，WaNet 的攻击性能对其核心参数高度敏感。$s < 0.5$ 时 warping 形变过弱，ASR 显著下降；$k < 4$ 时网格分辨率不足，后门信号被模型当作噪声忽略。这一狭窄的有效参数区间表明，WaNet 虽然有效，但并非一个"即插即用"的攻击方案，攻击者需要精心选择 warping 参数。

\section{结论}
% ============================================================

本文围绕 WaNet 后门攻击进行了系统的复现、分析与扩展实验。主要发现如下：

\begin{enumerate}[label=(\arabic*),leftmargin=1.5cm]
    \item WaNet 在 CIFAR-10 上实现了 99.15\% 的攻击成功率，同时保持 92.84\% 的干净准确率，且能绕过 STRIP、Fine-Pruning 和 Neural Cleanse 三种防御，验证了原论文的结论。
    \item Noise Mode 是 WaNet 绕过防御的决定性因素——关闭后 Cross Accuracy 降至零，模型学到的是可被检测的像素级捷径而非真正的 warping 变换。
    \item WaNet 对 warping 参数高度敏感，$s < 0.5$ 或 $k < 4$ 时攻击效果大幅下降，存在一个狭窄的有效参数区间。
    \item 从频域角度分析了 warping 操作在图像信号层面留下的痕迹，设计并实现了频域 warping 检测器，取得了 AUC 0.735 的检测性能，显著优于 STRIP 的 0.50。
\end{enumerate}

本文的主要创新在于：不同于现有防御方法（STRIP、Neural Cleanse 等）关注模型行为（输出熵、梯度、神经元激活），我们从图像信号本身的角度出发，利用 warping 操作依赖插值这一特性，在频域中找到了可检测的痕迹。这为检测 warping 类后门攻击提供了一个新的视角。
